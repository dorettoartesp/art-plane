# Offboarding de Usuário

Procedimento inicial para remover acesso de usuário em ambientes ARTESP.

## Checklist

```text
[ ] Identificar usuário e e-mail institucional
[ ] Revogar acesso no IdP, quando existir
[ ] Desativar/remover usuário no Plane
[ ] Revogar tokens pessoais/API
[ ] Reatribuir tarefas críticas
[ ] Registrar evidência do offboarding
```

Enquanto SSO/OIDC não estiver implementado, revisar usuários diretamente no painel
administrativo do Plane.
